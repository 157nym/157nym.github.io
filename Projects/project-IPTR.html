I.P.T.R

Inter Planetary Trade Route

Description

I.P.T.R is a solo game I made in first year in 3 days for a GameJam hosted by ICAN

The theme was Space Opera, I got to make everything for this game from the design, visuals and programming.
I made this game while learning how to use unity, this project helped me understand all the basic tools of the engine, as well as starting with C# for unity.
